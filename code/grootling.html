Grootling()

{

Description

Grootling() is a novel method for giving houseplants a voice and the ability to tweet their most basic needs and deepest desires in a desperate yet pathetic cry for attention.

An arduino is connected to a moisture sensor placed in the soil of a potted plant. The arduino logs the incoming soil moisture data, and looks for continued periods of moisture reports below a certain threshold. If the plant appears in peril, or simply needs watering, the arduino connects to the home wi-fi network and sends a signal to a host computer running a Python script. The python script then connects to Twitter and posts a tweet stating the plant's need.

Future Work

Due to the memory limitations of the arduino, and the need for wi-fi connectivity, a RaspberryPi would be a simpler hardware solution, and would bypass the need for connecting to a host computer, as well as greatly simplify the network code.
